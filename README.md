# ☕ Café Maná

Sitio web oficial de **Café Maná** - Café artesanal de montaña de San José de Ocoa, República Dominicana.

## 🌟 Características

- ✅ React + TypeScript + Vite
- ✅ Framer Motion para animaciones suaves
- ✅ Diseño responsive
- ✅ Paleta de colores "Raíces de Ocoa"
- ✅ Optimizado para rendimiento

## 🚀 Instalación

```bash
npm install
```

## 💻 Desarrollo

```bash
npm run dev
```

## 🏗️ Build para Producción

```bash
npm run build
```

## 📦 Preview

```bash
npm run preview
```

## 🌐 Despliegue

Este proyecto está configurado para desplegarse en Vercel.
